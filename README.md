# Multi-Site-OSPF-Enterprise-Network-Lab-GNS3-
A comprehensive GNS3 simulation of a multi-site enterprise network featuring: - OSPF dynamic routing across 3 routers - Frame Relay &amp; HDLC WAN encapsulation - Inter-VLAN routing across 10 VLANs - DHCP IP assignment for all end devices - Full end-to-end connectivity verified via pings and Wireshark
